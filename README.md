# BeautyScope

## 프로젝트 개요
BeautyScope는 화장품 브랜드와 제품 정보를 제공하는 웹 애플리케이션입니다. 이 프로젝트는 Flask를 사용해 후단 API를 구축하고, Firebase Firestore를 통해 브랜드 및 제품 데이터를 저장하며, 프론트엔드와의 연동을 지원합니다. 현재 중간 점검 단계에서 샘플 데이터(2개 브랜드, 2개 제품)를 사용 중이며, 향후 더 많은 데이터를 추가할 계획입니다.
